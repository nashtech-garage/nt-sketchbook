Get started any way you want

Use the package from npm, or load the compiled files from the generated dist folder. The page itself is built using these same assets.

Install with npm

Pull the package into an app and import the compiled CSS and JavaScript entry points.

Use compiled files

Reference dist assets directly from plain HTML when a simple static integration is enough.

Choose a theme

Start with the light or dark token set, then add NashTech brand styles on top.

Customize with tokens and themes

NT Stylesheet exposes CSS variables for colors, surfaces, borders, spacing, and component-level behavior. Override the variables where your product needs a local adjustment.

<link rel="stylesheet" href="themes/nt-theme-light.css" />
