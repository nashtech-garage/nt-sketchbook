Border utilities

From _border.scss. Use these classes to add borders, set border widths, choose semantic border colors, and control border radius.

Class pattern Result
nt-border Adds a 1px neutral border.
nt-border-top, nt-border-end, nt-border-bottom, nt-border-start Adds a border to one side.
nt-border-0 to nt-border-5 Sets border width from 0 to 5px.
nt-border-primary, nt-border-success, nt-border-danger Applies semantic border colors.
nt-rounded, nt-rounded-0 to nt-rounded-3 Applies base or sized radius.
nt-rounded-circle, nt-rounded-pill Creates circular or pill shapes.
