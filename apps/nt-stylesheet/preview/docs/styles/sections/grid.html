Grid system

From _grid.scss. Use nt-container, nt-row, and 12-column classes to build responsive layouts.

Class pattern Result
nt-container Centers content and applies responsive max widths.
nt-row Creates a wrapping flex row with column gutters.
nt-col-1 to nt-col-12 Sets fixed column spans on all screen sizes.
nt-col-sm-6, nt-col-lg-4 Sets responsive column spans at breakpoints.
