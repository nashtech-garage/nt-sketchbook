Flex utilities

From _flex.scss. Flex utilities are responsive. Add breakpoint names like sm, md, lg, xl, or xxl between the utility name and value.

Class pattern Values
nt-d-flex, nt-d-md-flex flex, inline-flex
nt-flex-row, nt-flex-md-col row, row-reverse, col, col-reverse
nt-justify-between, nt-justify-center start, end, center, between, around, evenly
nt-items-center, nt-self-end start, end, center, baseline, stretch
nt-flex-wrap, nt-gap-3 Wrap controls and gap scale from 0 to 6.
nt-order-first, nt-order-md-2 Order values 0 to 5 plus first and last.
