Utility documentation

Use and customize NT style utilities

These docs explain the compiled utility classes and the Sass modules behind them: border, animation, flex, grid, size, spacing, text, typography, and shared utilities.

Usage

In plain HTML, load the compiled stylesheet, theme, brand layer, and optional JavaScript bundle. The utility classes are available after css/nt.css is loaded. 

<link rel="stylesheet" href="css/nt.css" />
<link rel="stylesheet" href="themes/nt-theme-light.css" />
<link rel="stylesheet" href="brands/nashtech.css" />
<link rel="stylesheet" href="css/nt-icons.css" />
<script src="scripts/nt.js"></script>
Use utility classes for layout and small style adjustments. Use component classes like nt-button, nt-card-default, and nt-alert when the UI element already exists in the design system.

Border utilities

From _border.scss. Use these classes to add borders, set border widths, choose semantic border colors, and control border radius.

Class pattern Result
nt-border Adds a 1px neutral border.
nt-border-top, nt-border-end, nt-border-bottom, nt-border-start Adds a border to one side.
nt-border-0 to nt-border-5 Sets border width from 0 to 5px.
nt-border-primary, nt-border-success, nt-border-danger Applies semantic border colors.
nt-rounded, nt-rounded-0 to nt-rounded-3 Applies base or sized radius.
nt-rounded-circle, nt-rounded-pill Creates circular or pill shapes.
Primary bordered panel

This panel uses nt-border, nt-border-primary, and nt-rounded-3. 

<div class="nt-border nt-border-primary nt-rounded-3 nt-p-4">
    Primary bordered panel
</div>

Animation helpers

From _animation.scss. This module currently exposes Sass mixins and keyframes used by components. The mixins are intended for SCSS authors rather than direct HTML utility classes.

Mixin or keyframe Use case
nt-expand-collapse($duration, $timing) Creates hidden and nt-expanded states for collapsible content.
fade-in and fade-in mixin Fades an element in while moving it upward into place.
pulse and animate-pulse mixin Creates repeated opacity pulse behavior. 
@use '@nashtech-garage/stylesheet/src/styles/animation' as animation;

.product-panel {
    @include animation.nt-expand-collapse;
}

Flex utilities

From _flex.scss. Flex utilities are responsive. Add breakpoint names like sm, md, lg, xl, or xxl between the utility name and value.

Class pattern Values
nt-d-flex, nt-d-md-flex flex, inline-flex
nt-flex-row, nt-flex-md-col row, row-reverse, col, col-reverse
nt-justify-between, nt-justify-center start, end, center, between, around, evenly
nt-items-center, nt-self-end start, end, center, baseline, stretch
nt-flex-wrap, nt-gap-3 Wrap controls and gap scale from 0 to 6.
nt-order-first, nt-order-md-2 Order values 0 to 5 plus first and last. 
<div class="nt-d-flex nt-flex-wrap nt-items-center nt-justify-between nt-gap-3">
    <button class="nt-button nt-button-primary">Save</button>
    <button class="nt-button nt-button-outline-primary">Cancel</button>
</div>

Size utilities

From _size.scss. Width and height helpers cover common percentages, auto sizing, max sizing, and viewport sizing.

Class pattern Result
nt-w-25, nt-w-50, nt-w-75, nt-w-100, nt-w-auto Sets width to percentage or auto.
nt-h-25, nt-h-50, nt-h-75, nt-h-100, nt-h-auto Sets height to percentage or auto.
nt-mw-100, nt-mh-100 Constrains max width or height to 100%.
min-vw-100, min-vh-100, vw-100, vh-100 Uses viewport width and height units.

Spacing utilities

From _spacing.scss. The spacing scale runs from 1 to 8, starting at 0.25rem and increasing by 0.25rem.

Class pattern Result
nt-m-4, nt-p-4 Margin or padding on all sides.
nt-mt-2, nt-pr-3, nt-mb-6, nt-pl-5 Top, right, bottom, and left spacing.
nt-mx-4, nt-px-4 Horizontal left and right spacing.
nt-my-4, nt-py-4 Vertical top and bottom spacing. 
<section class="nt-mt-6 nt-px-4 nt-py-5">
    Use spacing utilities to keep layout rhythm consistent.
</section>

Text utilities

From _text.scss. These classes cover text alignment, wrapping, transformation, decoration, semantic text presets, font weight, style, size, and line height.

Alignment

nt-text-start, nt-text-center, nt-text-end, plus responsive variants such as nt-text-md-center.

Wrapping

nt-text-wrap, nt-text-nowrap, nt-text-truncate, and nt-text-break.

Transform and decoration

nt-text-uppercase, nt-text-capitalize, nt-text-decoration-underline, and nt-text-decoration-none.

Weight and style

nt-fw-regular, nt-fw-semibold, nt-fw-bold, nt-fst-italic. 

<p class="nt-text-md-center nt-text-para-md nt-fw-semibold">
    Responsive centered paragraph text.
</p>

Typography utilities

From _typography.scss and _utilities.scss. Mulish font faces are registered, nt-font applies the default font family, and generated text-size utilities follow the NT type scale.

Class pattern Result
nt-font Applies the default NT font family.
nt-text-4xs to nt-text-5xl Generated font-size and line-height utilities.
nt-line-height-4xs to nt-line-height-5xl Generated line-height utilities.
nt-fs-1 to nt-fs-6 Heading-like font sizes from 2.5rem to 1rem. 
<h2 class="nt-font nt-text-5xl nt-line-height-5xl">
    Product analytics
</h2>

Shared Sass utilities

From _utilities.scss. This module defines the type scale and reusable mixins used by typography and components.

Utility Purpose
font-size($size, $with-line-height) Applies a generated font size, optionally with line height.
generate-font-size-utilities Creates nt-text-* utility classes.
generate-line-height-utilities Creates nt-line-height-* utility classes.

Customize with Sass

The compiled package gives consumers the ready-to-use utility classes. When working inside this app, customize the Sass maps and mixins before running the build.

Change spacing scale

Edit $nt-space-scale in src/styles/_spacing.scss to add, remove, or resize spacing steps.

Change flex breakpoints

Edit $breakpoints in src/styles/_flex.scss when utility generation needs a different responsive scale.

Change grid containers

Edit $nt-container-max-widths in src/styles/_variables.scss to tune container widths.

Change type scale

Edit $nt-font-sizes or scale-font-size() in src/styles/_utilities.scss. 

// Example inside this app
$nt-space-scale: (
    '1': 0.25rem,
    '2': 0.5rem,
    '3': 0.75rem,
    '4': 1rem,
    '5': 1.25rem,
    '6': 1.5rem,
    '7': 1.75rem,
    '8': 2rem
);

// Then rebuild
// pnpm build:stylesheet-lib

Generated as static HTML and styled with NT Stylesheet itself.
