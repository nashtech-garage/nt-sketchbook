How to work with NT Stylesheet

Use this page as the main guide for installing the library, loading the generated files, applying themes, using utility styles, copying component examples, and finding icon classes.

Install

Add the package to your app when you want to consume the compiled stylesheet, icon font, themes, brand file, and optional JavaScript. 

npm install @nashtech-garage/stylesheet

Use dist files

The generated dist folder contains the files a plain HTML page needs. Load the core CSS first, then theme, brand, icons, and script. 

<link rel="stylesheet" href="css/nt.css" />
<link rel="stylesheet" href="themes/nt-theme-light.css" />
<link rel="stylesheet" href="brands/nashtech.css" />
<link rel="stylesheet" href="css/nt-icons.css" />
<script src="scripts/nt.js"></script>
File Purpose
css/nt.css Core reset, layout utilities, component styles, and base classes.
themes/nt-theme-light.css CSS variables for the active visual theme.
brands/nashtech.css NashTech brand color and identity overrides.
css/nt-icons.css Icon font classes using the nti prefix.
scripts/nt.js Optional behavior for interactive components.

Theme and brand

Set the theme file in the page head and use the matching value on data-theme. Add the brand stylesheet after the theme so brand variables can refine the base tokens. 

<body data-theme="nt-theme-light">
    ...
</body>

Style modules document the utility classes used for layout, spacing, type, sizing, borders, animation, and helper behavior.

Border and size

Border radius, border width, sizing, width, height, and max-width helpers.

Spacing and layout

Margin, padding, flex, grid, and layout utilities for page composition.

Text and typography

Text alignment, color, weight, headings, and body copy styles.

Utilities

General helpers for visibility, display, state, and common UI needs.

Sass custom

Customize from the Sass source when you need deeper control over modules like _border.scss, _animation.scss, _flex.scss, _grid.scss, _size.scss, _spacing.scss, _text.scss, _typography.scss, and _utilities.scss.

JavaScript behavior

Include scripts/nt.js when a page uses interactive components such as modal, dropdown, tooltip, popover, collapse, combobox, multiselect, or date picker. 

<button
    class="nt-button nt-button-primary"
    data-modal-target="#demo-modal"
>
    Open modal
</button>
